# 알림톡 템플릿 목록 (뿌리오 등록용)

2027 컨설팅 예약 시스템에서 사용하는 카카오 알림톡 템플릿 모음입니다.  
각 템플릿을 **뿌리오 콘솔**(`https://www.ppurio.com`)에서 등록한 뒤, 검수 완료되면
관리자 페이지 → **알림톡** 탭에서 `템플릿 코드`와 `changeWord 매핑(JSON)` 을 입력해주세요.

## 공통 안내

- 변수는 모두 `#{변수명}` 형식입니다 (뿌리오 표준).  
- 뿌리오 템플릿 본문에는 `#{변수명}` 그대로 적고, `changeWord` 에 애플리케이션 토큰을 매핑합니다.  
- 각 템플릿 끝에는 버튼(채널추가 / 웹링크 등)을 원하는 대로 함께 등록해주세요.
- 관리자 알림용 템플릿은 학생 이벤트 발생 시 **관리자 휴대폰 번호**로 발송됩니다.  
  관리자 알림 수신 번호는 알림톡 설정 탭의 `관리자 수신번호` 항목에서 관리합니다.

## 사용 가능한 애플리케이션 토큰

| 토큰 | 의미 |
|---|---|
| `${name}` | 학생 이름 |
| `${school}` | 학교명 |
| `${grade}` | 학년 (고3/고2/고1) |
| `${seat}` | 좌석 번호 |
| `${dateLabel}` | 기본 예약 일자 (예: "6월 5일 (금)") |
| `${slot}` | 기본 예약 시간 (예: "10:00") |
| `${currentDateLabel}` `${currentSlot}` | 현재 예약(변경/취소 맥락에서 명시적 값) |
| `${newDateLabel}` `${newSlot}` | 변경 희망 일시 |
| `${offeredSlots}` | 관리자가 제시한 시간대들 (`"6월 6일 14:00 / 6월 7일 10:00"` 형태) |
| `${reason}` | 사유 (학생 사유 또는 관리자 반려 사유) |
| `${accountId}` | 학생 로그인 아이디 |
| `${accountPw}` | 학생 초기 비밀번호 |

# I. 학생에게 발송되는 템플릿

## 1. `bookingComplete` — 예약 완료

**본문**
```
[2027 컨설팅] 상담 예약 완료 안내

#{학생명}님, 상담 예약이 정상적으로 완료되었습니다.

▶ 예약 일시: #{일시}

변경 또는 취소가 필요한 경우 마이페이지에서 신청해 주세요.
예약 시간 10분 전까지 도착 부탁드립니다.
```

**changeWord 매핑**
```json
{
  "#{학생명}": "${name}",
  "#{일시}": "${dateLabel} ${slot}"
}
```

---

## 2. `changeRequest` — 예약 변경 요청 접수

**본문**
```
[2027 컨설팅] 예약 변경 요청 접수

#{학생명}님의 예약 변경 요청이 접수되었습니다.

▶ 현재 예약: #{현재일시}
▶ 희망 일시: #{희망일시}
▶ 요청 사유: #{사유}

관리자 확인 후 승인 또는 반려 결과를 다시 안내드립니다.
```

**changeWord 매핑**
```json
{
  "#{학생명}": "${name}",
  "#{현재일시}": "${currentDateLabel} ${currentSlot}",
  "#{희망일시}": "${newDateLabel} ${newSlot}",
  "#{사유}": "${reason}"
}
```

# II. 관리자에게 발송되는 템플릿

관리자 알림은 `settings/ppurio.adminPhones` 배열에 등록된 모든 번호로 발송됩니다.

## 12. `adminNotifyBooking` — 학생 예약 완료 알림

**본문**
```
[2027 컨설팅/관리자] 신규 예약

#{학생정보} 학생이 상담 예약을 완료했습니다.

▶ 예약 일시: #{일시}

관리자 페이지에서 확인해 주세요.
```

**changeWord 매핑**
```json
{
  "#{학생정보}": "${name}(${grade}/${school})",
  "#{일시}": "${dateLabel} ${slot}"
}
```

# 등록 순서 권장

1. 뿌리오 콘솔에서 위 16개 템플릿 본문을 각각 새 템플릿으로 등록 (검수 요청).  
2. 검수 완료 후 각 템플릿에 부여된 **템플릿 코드**를 복사.  
3. 관리자 페이지 → 알림톡 탭에서 각 이벤트별 템플릿 코드와 changeWord JSON 을 입력 → 저장.  
4. 관리자 수신번호(`adminPhones`) 에 학원 관리자 휴대폰 번호 등록.  
5. "테스트 발송" 으로 각 템플릿 본인 번호에 확인.  
6. 실제 학생 계정으로 예약 → 전체 자동 발송 플로우 검증.
